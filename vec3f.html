Vec3f¶

A vector class with 3 floating point components.

Example Usage

>>> from cxxpythondocs import Vec3f
>>> vec = Vec3f(1, 2, 3)
>>> print(vec.x, vec.y, vec.z)
1.0 2.0 3.0
>>> vec.DotProduct(Vec3f(-1, -1, 1))
0.0
